# Cenas

Criar uma classe abstrata "Cena" que conterá todos os comportamentos que cenas devem ter
Criar cada cena como herdeira desta

Cenas fixas (História e loja) serão instanciadas diretamente na classe Main
Cenas aleatórias seguirão uma lógica utilizando um método dentro da classe Cena que manterá
tracking de um array contendo referências para cada classe aleatória e quais já foram instanciadas
para evitar repetições
